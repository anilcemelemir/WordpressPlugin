A plugin to manage events with a custom post type and taxonomy.

== Description ==

Event Manager allows you to create and manage events on your WordPress site.
It registers a custom post type named "Event" and a custom taxonomy named "Event Type".

Features:

* Custom Post Type: Event
* Custom Taxonomy: Event Type
* Full REST API support (show_in_rest = true)
* Internationalization (i18n) ready
* Gutenberg compatible

== Installation ==

1. Upload the `event-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Events' menu in the admin dashboard.
